Count Proper Words

  • Ask Doubt
  • Time-Limit: 2 sec
  • Score: 100
Difficulty :
  • Memory: 256 MB
  • Accepted Submissions: 100
Relevant For:AZ-201

Description

A proper word is a string made of lowercase English alphabets and contains at least 1 vowel and at least 1 consonant.

You need to find how many proper words of length N exist. As, the number can be very large, print it as modulus 1000000007.

Note - a,e,i,o,u are considered as vowels, and the rest of the alphabets are considered as consonants.

Input Format

The first line contains an integer T (number of test cases), 1 <= T <= 10^5.

The second line of each test case contains an integer N, 1 <= N <= 10^15.

Output Format

For each test case output a single integer on a new line, the total number of proper words of length N modulo 1000000007.

Sample Input 1 Copy

2
   2
   3

 

Sample Output 1 Copy

210
   8190
